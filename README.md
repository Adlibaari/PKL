# Deteksi Gerobak Pedagang Kaki Lima
## Dataset
Dataset didapatkan dari [Roboflow](https://universe.roboflow.com/barry-aprtz/foodcart-dnty6) yang memiliki gambar gerobak atau food cart yang bervariasi.

## Environment
- Quadro RTX 5000
- Python 3.12.1
- Pytorch 2.5.1
- Torchvision 0.20.1
- Torchaudio 2.5.1
- Ultralytics 8.3.39
